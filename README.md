Project Name: TaskFlow Landing Page
Developer: mohammad ameen

1. Technology Stack Used:
   - HTML5
   - CSS3 (Flexbox & CSS Grid)
   - JavaScript (Vanilla ES6)

2. Reason for Choice:
   I chose to use plain HTML/CSS/JS without frameworks (like React) for this specific task because:
   - Performance: It ensures the fastest possible load time for a simple landing page.
   - Simplicity: It keeps the codebase clean and lightweight without unnecessary dependencies.
   - Control: It demonstrates a strong understanding of the core web technologies and DOM manipulation.

3. Project Structure:
   / (Root)
   ├── index.html      # Main structure and semantic markup
   ├── style.css       # Styling using modular CSS, Variables, and Media Queries
   ├── main.js         # Navigation logic and animations
   └── README.txt      # Project documentation
